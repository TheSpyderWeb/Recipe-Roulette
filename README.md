# Recipe Roulette

**Recipe Roulette** is a web application designed to make discovering new recipes fun, interactive, and personalized. Users can search for recipes, save favorites, and even chat with a built-in AI assistant, ChefBot, to get recipe suggestions or food tips.  

This project combines frontend development with Firebase authentication and Firestore to provide a secure, full-featured experience.

---

## Features

- **Random Recipe Generator:** Get a new recipe every time you visit the homepage.  
- **Search Recipes:** Search for recipes by ingredients or keywords using the [MealDB API](https://www.themealdb.com/api.php).  
- **Favorites:** Save your favorite recipes and quickly access them later.  
- **Chatbot - ChefBot:** Ask for recipe recommendations, cooking tips, or fun food facts.  
- **User Authentication:** Sign up and log in with email/password using Firebase Auth.  
- **User Profiles:** Display user-specific information such as username and email.  
- **Responsive Design:** Optimized for desktop and mobile users.  

---

## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+), Lottie animations  
- **APIs:** [MealDB API](https://www.themealdb.com/api.php) for recipe data  
- **Authentication & Database:** Firebase Authentication and Firestore  
- **Styling & Layout:** Flexbox, custom CSS for responsive design
